# Raptor Grid Draw 12 players (Full Metal Plate 1.7)
Build grid base on size 12 which is efus size

- Corner base size 16x16, mid base size is 14x16, the most fair and largest distribution without blocking the corner. Possible to do 15x16 but most corner antisocial are inflexible on building outwards from corner and mid base too vulnerable to unlucky early hive spawn
- Lines are drawn in smaller parts so zooming in works and easier to erase draw points for opening
- Lines are drawn by ticks to avoid spam protection
- Widget will run on first load and disable itself after drawing
- Widget is disabled when spectating
- Widget is disabled when not playing raptor or scavenger
- Widget is disabled when not on Full Metal Plate 1.7
