# Build Menu / Grid Menu with Costs/Second

A dynamically patched version of the built-in Grid Menu that adds Costs per Second and Build Time information.

Both within the Grid Menu and under the Cursor.

In order:
Metal Cost / second
Energy Cost / second
Build Time

Costs per Second and Build Time automatically update with the amount of Selected Build Power

Display of info in Build Menu and Info under Cursor can be toggled independently
Decimal values for Build Time can be toggled on and off. Off by default.
