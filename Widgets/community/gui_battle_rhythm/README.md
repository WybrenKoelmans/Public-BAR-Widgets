# BAR: Battle Rhythm — Coaching Widget

This widget is inspired by “In-Game Coach for Beginners” by @uBdead, which does a great job helping new players learn the basics.
I wanted to take that idea further and build something calmer and smarter, aimed at guiding mid → high level decision-making, without being spammy or restrictive.

Note: This widget provides guidance and context, not automation — all decisions remain yours.

Battle Rhythm helps guide decisions around:
• economy flow
• tech timing
• production tempo
• build priorities
• converting advantage into pressure

What it does differently from beginner coaching:
• Uses net eco (income vs spend), not just resource bars
• Correctly handles Energy → Metal conversion
• Distinguishes real stalls from normal project drain (AFUS, reactors, tech)
• Detects game phases (Opening, T2 Spike, Tech Stress, Pressure, T3 Endgame)
• Gives tempo guidance only when eco is strong
• Encourages converting eco advantage into pressure, not just more economy

It’s meant to be a next-step coaching overlay, not a replacement for “In-Game Coach for Beginners”
